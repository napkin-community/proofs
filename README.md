napkin-proofs
========
Lean4 formal proofs of "An Infinitely Large Napkin".

```bash
# Build the project
lake build

# Type-check specific files in CLI
lake env lean NapkinProofs/Obviouslib.lean
lake env lean NapkinProofs/Chapter1.lean
```
